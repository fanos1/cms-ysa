This is a basic content management system where the end user can change some of the website's content via admin panel

---------------------------------------------------------------------------------
As an example, the following website is using this CMS : http://www.ysadrivingschool.com/
--------------------------------------------------------------------------------------------

This CMS has 2 MYSQL connection files:
1 The pdo-connection.php file is for the FRONT END pulling data from the DB,
2 And the mysqli-connection.php file is used in the /admin/ side of the website
